这是一个基于动态时间规整（DTW）算法的签名认证系统，主要功能如下：

1.用户注册与签名录入
用户输入用户名，系统自动生成唯一用户ID。
在画布上用手写板/鼠标进行签名，系统记录签名的轨迹点（坐标、时间戳、压力等）。
将签名数据存入MySQL数据库（user表和signatures表）。

2.签名验证
输入用户名和ID，在画布上书写待验证的签名。
系统从数据库中取出该用户已注册的参考签名。
利用DTW算法计算当前签名与参考签名之间的相似度，并与预设阈值（默认0.8）比较，输出“通过”或“失败”。

3.特征提取与对齐
从签名轨迹中提取多维特征：位置（x, y）、速度、方向角、加速度等。
将签名起点平移到原点（位置对齐），并进行Z-score归一化和重采样至相同长度，消除书写位置、大小、速度差异的影响。

4.可视化对比
左侧子图显示当前签名与参考签名的轨迹（起点对齐）。
右侧子图显示DTW最优匹配路径，并标注相似度分数和验证结果。

5.用户管理
查看所有已注册用户及其签名数量。
删除指定用户及其所有签名数据。
