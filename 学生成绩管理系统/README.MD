本系统基于python语言和MySQL实现学生成绩管理系统，系统提供了三种角色身份，分别是管理员，教师，学生。

1. 学生（权限等级 1）
选课：根据当前日期，查看可选课程安排（选课ID、课程ID、教师ID），选择后录入选课表。
退课：查看已选课程，删除指定选课记录。
成绩查询：
	查询单科成绩（按选课ID）
	查询全部成绩及学籍分（Endscore）

2. 教师（权限等级 2）
评分：在评分截止日期前，为所教班级的学生录入成绩。
成绩表查询：
	按课程教学班ID查看学生成绩
	按班级ID查看学生成绩
数据统计：
	按班级统计平均分、最高分、最低分、及格率
	按课程统计上述指标
	统计某学生的平均成绩
输出学生名单：调用存储过程，按班级输出学生ID、姓名、学籍分。

3. 教务管理员（权限等级 3）
基础数据维护（增、删、查）：
	学生表（学生ID、姓名、班级ID）
	教师表（教师ID、姓名）
	班级表（班级ID、班级名称）
	课程表（课程ID、课程名称、学分）
	选课安排表（选课ID、课程ID、教师ID、选课起止时间、评分截止时间、限选人数）
数据库备份与恢复：通过 mysqldump 和 mysql 命令实现备份（.sql 文件）与恢复。
