这是一个文件加密与解密工具，基于 cryptography 库的 Fernet 对称加密算法实现，主要功能包括：

1.文件加密（导入）
用户指定一个待加密的普通文件路径，以及加密后文件的存放目录。
程序使用当前有效的密钥对文件内容进行加密，加密后将原文件移动到加密目录（覆盖原位置）。
输出加密后的文件（不可直接读取）。

2.文件解密（导出）
用户指定一个已加密的文件路径，以及解密后的文件存放目录。
程序使用当前密钥尝试解密，成功后将解密文件移动到指定目录，原加密文件随之移动。
若密钥错误或文件损坏，提示解密失败。

3.口令管理（支持多版本回滚）
首次运行时，要求用户输入初始口令，系统生成一个随机加密密钥，并将口令的 SHA‑256 哈希值以及密钥（Base64 编码）保存在 password.txt 文件中。
修改口令：验证旧口令正确后，生成新的随机密钥，并将新口令哈希和新密钥追加到文件中（旧密钥保留）。
恢复口令：当忘记新口令或想回退时，可以恢复到上一个口令和对应的旧密钥（即删除最新一条记录）。
退出程序时自动删除 password.txt 文件，不留痕迹。
